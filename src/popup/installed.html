New/Hidden Features

  • Preview notes on mouseover:

    • If you display the xnotes column and widen it, so that the three dots are shown for a note, but no text: then, on mouseover/hover, a little window will show the note text. The message does not need to be selected. This can be used to get a fast overview over all notes indicated in the messagelist. The number of characters shown can be chosen in the settings (tools->addon settings-> xnote++).
